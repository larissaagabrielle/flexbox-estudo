Order

Propriedade que permite mudar a ordem dos itens. Por padão, o valor do order é 0, com isso, ele segue a ordem que foi declarada na HTML. Quando se declara valores para essa propriedade, há a mudança de ordem.

Order: 0;

1
2
3

Order: 1;

Quando coloca valor 1 para a propriedade, o item ficará atrás de todos os elementos com order: 0; Como é o caso do bloco 2, com order: 1; ele passa a ficar trás dos outros elemetos com order: 0;

1
2
3

Order: -1;

Com valores negativos, o elemento passa a ficar à frente dos elementos com valores positivos. Como é o caso do bloco 3.

1
2
3

Order: -1 0 1

1
